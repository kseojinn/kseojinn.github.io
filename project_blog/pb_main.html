Project Blog

Welcome to my Project Blog! Here, I will update on the various projects I am working on.

To systematically and flawlessly bring the overflowing ideas in my mind into reality, I started the WbW (Week by Week) Project Routine from May 16th. I put deep thought into ensuring that every project is executed with the highest level of quality and completion.

  • Mon ~ Fri: Develop the Project

  • Sat ~ Sun: Wrap-Up the Project & Map out the next Project

WbW (Week by Week) Project

  • [May W3] SoFa: Seek Your Fit Lab

    • SoFa utilizes Sentence-BERT based semantic similarity analysis to recommend laboratories that align with a user's research interests. By embedding and comparing textual data such as research topics, paper abstracts, and laboratory descriptions, the system can identify relevant laboratories even when exact keywords do not match.

    • The motivation for this project came from the concern that valuable opportunities could be overlooked when searching for laboratories manually.

    • The inspiration for this project came from learning that transformer-based models such as BERT can be fine-tuned to capture semantic relationships between texts.

Projects

  • SFED: SLMs for Edge Devices.[Github]

    • The goal of this project is to experiment with hardware bottlenecks and quantization efficiency when deploying Small Language Models (SLMs) in resource-constrained edge computing environments.

    • The Sweet Spot is Q4_K_M quantization. Because of Memory bandwidth defense and Practical computational throughput (TPS)

    • Qwen3-1.7B demonstrated the most outstanding deployment efficiency in resource-constrained edge environments.

  • Validation of Zipf's Law and Heap's Law Using Texts from websites.[Github]

    • This project verified Zipf's law and Heap's law using text collected from actual websites (The New York Times, The Guardian, and NHK).

    • Despite differences in language, both English and Japanese followed Zipf's Law and Heaps' Law. Frequently appearing words existed in both languages, and as more text was added, the rate of new word appearances gradually decreased. Due to strict anti-crawling policies on news websites, the project was conducted on a small-scale dataset using only a single article page from each source.

  • Weather Forecasting Code Using Regression Models.[Github]

    • This project is a simple hands-on exercise in weather forecasting using linear regression and Lasso regression models.

  • Low-cost, Low-power Road Icing Detection System (Graduation Project).

    • Leading the development of an efficient edge-based system for road icing detection utilizing LoRa communication, responsible for the AI model and overall system design

    • Collaborators: Jaeho Lee, Kyeongjun Park, Donghyun Park

  • AI Image Studio: Context-Preserving Multi-Style Image Transformation System.[Github]

    • To overcome the compositional limitations of text-based generative models, developed a web platform that preserves the position and context of user-placed objects while transforming their style. Built an intuitive compositional design interface using Fabric.js and a real-time object background removal system via RemBG. Through the FLUX Dev model and prompt engineering, implemented an animation style transformation pipeline for eight styles: Studio Ghibli, Pixar, Frozen, Snow White, Cinderella, Beauty and the Beast, Aladdin, Retro.

    • 1st Place, AI API Competition, Dept. of Computer Engineering, Sungkyul University, Dec 2025

  • SKU Course Scheduler.[Github]

    • To support college students' course registration planning, which requires consideration of complex constraints, developed an automated timetable generation system based on AI optimization algorithms. Applied Genetic Algorithms(GA), Simulated Annealing(SA), and a hybrid optimization method combining both approaches. This enabled the design of four optimization types based on user preferences (avoiding morning classes, securing lunch breaks, maximizing free periods, and evenly distributing classes by day) along with essential/exclusion course settings, achieving personalized scheduling. It was built using actual lecture data from Sungkyul University (24 departments, all lectures except a few).

  • LLM-powered Voice Morning Briefing Assistant.[Github]

    • Developed a Raspberry Pi 5-based assistant system that delivers personalized information via voice every morning for busy modern individuals. By integrating with the Public Data Portal (weather) and Naver News API, the system collects information summarized by an LLM (Gemma-2b) and delivers a natural voice briefing through TTS.

    • 6th Place, AI API Competition, Dept. of Computer Engineering, Sungkyul University, Jun 2025

  • Design of a Baby Cry Prediction System.

    • To address the difficulty novice parents face in interpreting their baby's cries, designed an AI prediction system that analyzes the cause of crying and provides real-time alerts. Classified infant crying sounds into five types—hunger, sleepiness, false crying, discomfort, and pain—aiming to build customized prediction models for each cause.

    • 4th Place, Design Competition, Dept. of Computer Engineering, Sungkyul University, Nov 2023
